# Spotify Recommendation System

Projeto de Data Science focado na construção de um sistema de recomendação de músicas utilizando Machine Learning.

## Objetivo

Desenvolver um sistema que recomenda músicas semelhantes com base em características de áudio presentes no dataset do Spotify.

## Tecnologias previstas

- Python
- Pandas
- NumPy
- Scikit-Learn
- Matplotlib
- Git/GitHub
